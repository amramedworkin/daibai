# Northwind Instances


| Platform                 | Access Type | Features                                                                                          |
| ------------------------ | ----------- | ------------------------------------------------------------------------------------------------- |
| **SQL Online IDE**       | Web UI      | Allows you to select Northwind from a list of remote databases to query directly in your browser. |
| **PDBM Book Playground** | Web UI      | Offers a dedicated "classic Northwind database" playground for learning and testing queries.      |
| **PlayCode.io**          | WebAssembly | Runs a full SQL compiler in your browser with sample templates available for Northwind.           |
